## Simulators of Langmuir-Hinshelwood reactions in nanoparticle catalysis

### Reaction types
- without products
- with products
- mobile adsorbates
- immobile adsorbates

### Provided Interactions
- Lennard-Jones potential
- Shifted Lennard-Jones potential
- Screened coulomb interaction (Yukawa)
- Gaussian potential
